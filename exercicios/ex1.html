A Linguagem HTML

HTML (abreviação par a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação ultilizada na construção de páginas na web. Documentos HTML podem ser interpretados por navegadores.

Alguns dos recursos da linguagem HTML vistos até este momento :
Recursos Tags assiciadas Descrição
Links < a > Criar links entre páginas
Lista < ul > e < li > Cria lista de items ordenados
< ol > e < li > Criar lista de items não ordenados
Imagens < img > Inserir imagens
